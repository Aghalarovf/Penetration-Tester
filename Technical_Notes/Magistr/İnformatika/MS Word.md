## <a name="overview"></a>1. Overview & File Formats

| Format | Extension | Description |
|--------|-----------|-------------|
| Word Document | `.docx` | Default modern format (XML-based) |
| Word 97–2003 | `.doc` | Legacy binary format |
| Word Template | `.dotx` | Reusable template |
| Word Macro-Enabled | `.docm` | Contains VBA macros |
| PDF | `.pdf` | Export/read only |
| Rich Text Format | `.rtf` | Cross-platform text |
| Plain Text | `.txt` | No formatting |
| Web Page | `.htm / .html` | For web publishing |

**Zoom Range:** 10% – 500%  
**Max Font Size:** 1638 pt  
**Default Max Font Size:** 72 pt
**Min Font Size:** 1 pt  
**Default Min Font Size:** 8 pt
**Max Recent Files:** 50

## <a name="file-tab"></a>2. File Tab (Backstage View)

### Info
- **Protect Document** — Restrict editing, encrypt with password, mark as final, add digital signature
- **Inspect Document** — Check for hidden data, metadata, accessibility issues, compatibility
- **Manage Document** — Recover unsaved documents
- **Properties** — Title, author, tags, subject, category, comments

### New
- **Blank Document** — `Ctrl + N`
- **Templates** — Search online or local templates

### Open
- `Ctrl + O` — Open file dialog
- **Recent** — Shows last 50 documents
- **OneDrive / SharePoint** — Cloud locations
- **This PC** — Local folders

### Save / Save As
- `Ctrl + S` — Save
- `F12` — Save As dialog
- `Ctrl + Shift + S` — Save As (alternate)

### Print
- `Ctrl + P` — Print dialog
  - **Print All Pages** — Entire document
  - **Current Page** — Only active page
  - **Custom Range** — e.g., 1-3, 5, 7-9
  - **Odd Pages** — For double-sided printing
  - **Even Pages** — For double-sided printing
  - **Print Markup** — Include tracked changes
  - **Copies** — Number of copies
  - **Collated / Uncollated**
  - **Portrait / Landscape**
  - **Letter / A4 / Custom Paper Size**
  - **Margins** — Normal, Narrow, Moderate, Wide
  - **1/2/4 Pages per Sheet** — Multiple pages
  - **Print Preview** — Live preview on right side

### Share
- **Share with People** — Invite collaborators (OneDrive)
- **Email** — Send as attachment, PDF, or link
- **Present Online** — Share via browser

### Export
- **Create PDF/XPS** — Export to PDF
- **Change File Type** — Save in different formats

### Account
- **User Info** — Name, photo
- **Office Theme** — Colorful, Dark Gray, Black, White
- **Office Background**
- **Connected Services** — OneDrive, LinkedIn, etc.

### Options
- **General, Display, Proofing, Save, Language, Accessibility, Advanced, Customize Ribbon, Quick Access Toolbar, Add-ins, Trust Center**

## <a name="tips"></a>22. Tips & Tricks

### Productivity Tips
- **AutoCorrect** — File → Options → Proofing → AutoCorrect Options. Auto-fix typos, replace abbreviations.
- **AutoText** — Type abbreviation + `F3` to expand pre-saved text blocks.
- **Quick Parts** — Insert → Quick Parts → Save Selection to save reusable content.
- **Styles** — Always use heading styles for automatic TOC generation.
- **Format Painter** — Double-click to lock, apply to multiple areas, `Esc` to unlock.
- **Selection Pane** — Layout → Selection Pane to manage overlapping objects.
- **Read Mode** — Best for reviewing. Press `Esc` to return to editing.
- **Focus Mode** — View → Focus for distraction-free writing.
- **Immersive Reader** — View → Immersive Reader for accessibility features.
- **Drag and Drop** — Select text and drag. Hold `Ctrl` to copy.
- **Smart Lookup** — Right-click a word → Smart Lookup for web definitions.
- **Dictate** — Home → Dictate for voice typing (`Alt + `\`).
- **Document Inspector** — File → Info → Check for Issues → Inspect Document.

### Protect & Security
- **Encrypt with Password** — File → Info → Protect Document → Encrypt with Password
- **Restrict Editing** — File → Info → Protect Document → Restrict Editing
- **Mark as Final** — Marks as read-only, discourages editing
- **Digital Signature** — Add invisible/visible signature
- **Protected Documents** → File → Info → Protect → Encrypted with Password

### Print Settings
- **Print Markup** — Prints tracked changes
- **Print Background Colors** — File → Options → Display → Print background colors
- **Print Drawing Objects** — Includes shapes/images
- **Collated vs Uncollated** — Full sets vs page-by-page

### Mail Merge Quick Steps
1. Mailings → Start Mail Merge → Letters
2. Select Recipients → Use an Existing List (Excel/CSV)
3. Insert Merge Fields where needed
4. Preview Results
5. Finish & Merge → Print or Edit Individual Documents

### Document Recovery
- **AutoRecover** — File → Options → Save → AutoRecover every X minutes
- **Recover Unsaved** — File → Info → Manage Document → Recover Unsaved Documents
- AutoRecover location: `C:\Users\[name]\AppData\Roaming\Microsoft\Word\`

### Collaboration
- **Share** — File → Share (requires OneDrive)
- **Track Changes** — `Ctrl + Shift + E`
- **Comments** — `Ctrl + Alt + M`
- **Compare Documents** — Review → Compare
- **Combine Documents** — Review → Combine (merge tracked changes)

### View Modes Summary
| Mode | Best For |
|------|----------|
| Print Layout | Everyday editing |
| Read Mode | Reviewing without editing |
| Web Layout | Documents for web publishing |
| Outline | Reorganizing document structure |
| Draft | Fast text editing, no graphics |
| Focus | Distraction-free writing |
